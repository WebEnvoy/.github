# WebEnvoy

WebEnvoy 是面向 Agent 和上游系统的网站能力基础设施。

它将复杂网站的站点知识、站点能力、执行身份、任务封装和多通道执行策略组织成一个可演化的平台，让 Agent / 程序以更低探索成本、更稳定的执行方式完成网站读写任务。
